# C Learning Projects

C语言学习项目合集，包含数据结构、算法和系统编程练习。

## 项目列表

### 1. 链表操作 (LinkedList)
- **路径**: `C 链表操作/`
- **内容**: 双向链表、循环链表实现
- **核心功能**: 节点增删改查、链表反转、合并排序

### 2. 排序算法 (Sorting)
- **路径**: `C 排序算法/`
- **内容**: 冒泡排序、快速排序、归并排序、堆排序
- **核心功能**: 算法性能对比、可视化输出

### 3. 超市管理系统 (Supermarket)
- **路径**: `C语言超市管理系统/`
- **内容**: 商品管理、库存系统、销售记录
- **核心功能**: 文件持久化、数据查询、报表生成

### 4. AT指令系统 (AT_Command)
- **路径**: `AT小型指令系统/`
- **内容**: 嵌入式AT指令解析器
- **核心功能**: 串口通信、指令解析、状态机实现

## 技术栈

- **语言**: C11/C99
- **编译器**: GCC / MSVC
- **开发环境**: VSCode + Keil

## 编译运行

```bash
# 进入项目目录
cd "C 链表操作"

# 编译
gcc -o linkedlist main.c linkedList.c

# 运行
./linkedlist
```
